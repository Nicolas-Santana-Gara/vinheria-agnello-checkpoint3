Vinheria Agnello

Sistema de Gerenciamento de Vinhos

Sobre o Sistema

Este sistema foi desenvolvido para gerenciar o estoque de vinhos da Vinheria Agnello. Ele permite:

  • Adicionar novos vinhos ao estoque
  • Listar todos os vinhos disponíveis
  • Identificar vinhos com estoque baixo
  • Calcular o estoque total da vinícola
  • Exibir os nomes dos vinhos em diferentes formatos

Abra o console do navegador (F12) para visualizar as operações do sistema.
